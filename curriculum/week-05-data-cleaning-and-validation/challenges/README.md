# Week 5 — Challenges

Two open-ended challenges, meant for after you finish all three exercises. Where the exercises had a defined "expected result," these have judgment calls — read the rubric in each file before you start so you know what "good" looks like.

## How to work these

- Do the [exercises](../exercises/README.md) first — both challenges assume the dedupe, standardize, and dependent-drop-down muscle memory from Exercises 1–3.
- There's no single correct answer to either challenge. Document the decisions you made, the same way Week 1's challenges asked you to — this habit only gets more important as the data gets messier and the stakes get higher.
- Give each challenge its own new sheet or file, named clearly, so your work is easy to review.
