# Week 7 — Exercises

Three exercises, meant to be done in order, each 45–75 minutes. Together they turn the three lectures into muscle memory: building a pivot from scratch, grouping dates and reframing sums as percentages, and choosing (then building) the right chart for a question.

## How to work these

- Use the `Transactions` Table from the [week README](../README.md) as the source for every exercise. Build each pivot on its own new sheet, named for the exercise (e.g., `Ex1-Pivot`), so nothing overwrites another exercise's work.
- Do them **in order**. Exercise 2 assumes the plain pivot-building fluency from Exercise 1; Exercise 3 assumes the grouped/percentage pivots from Exercise 2 as chart sources.
- Work in whichever app is your primary this week, but build at least one pivot in the *other* app too — the goal is to notice exactly where the workflows diverge (grouping dialogs, calculated fields, Show Values As) while it's fresh.
- When a task gives an "Expected" number, check your pivot against it before moving on.
