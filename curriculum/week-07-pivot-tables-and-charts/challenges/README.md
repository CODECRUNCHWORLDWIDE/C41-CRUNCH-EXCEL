# Week 7 — Challenges

Two open-ended challenges, meant for after you finish all three exercises. Where the exercises had a defined "expected result," these ask for judgment — read each file's rubric before you start so you know what "good" looks like.

## How to work these

- Do the [exercises](../exercises/README.md) first — both challenges assume the pivot-building, grouping, and charting fluency from Exercises 1–3.
- Challenge 1 has right answers (check them), but the strategy for getting there — which pivot configuration you reach for — is your call.
- Challenge 2 is fully open-ended: there's no single "correct" fixed chart, only better and worse ones. Document the decisions you made, the same way you'll be asked to in later weeks when the stakes are higher.
- Give each challenge its own new sheet, named clearly, so your work is easy to review.
