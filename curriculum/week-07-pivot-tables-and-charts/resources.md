## A note on versions

This course targets **Excel 365 / Excel 2021 or newer** and the **current web version of Google Sheets**. Pivot table and chart menu paths described in the lectures (e.g., "PivotTable Analyze → Fields, Items & Sets → Calculated Field") match these versions. If your Excel is 2016 or earlier, the same features exist under a "PivotTable Tools → Options" ribbon tab instead of "PivotTable Analyze" — search the exact feature name in Help (`Alt+Q`) if a described menu path doesn't match what you see. Google Sheets' pivot editor has changed its exact layout more than once in recent years; if "Show as" or "Group by" isn't where a lecture describes, it's almost always still present under the same field's settings, just reorganized.
