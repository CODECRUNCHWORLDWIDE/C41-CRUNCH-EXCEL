# Week 1 — Resources

Curated links only — official documentation and the handful of tools actually worth installing this week. Everything here is free.

## Installing / accessing the tools

**Microsoft Excel**

- Included with a Microsoft 365 subscription (personal or family plans start free-trial, then paid). If you don't have a subscription, use **Excel for the web** free with any Microsoft account: <https://www.microsoft.com/en-us/microsoft-365/excel>
- Excel for the web has nearly all the formatting features this week needs (number formats, conditional formatting, freeze panes) — a paid desktop license is *not* required to complete Week 1.

**Google Sheets**

- Free with any Google account: <https://sheets.google.com>
- No install needed — runs entirely in the browser. Optional: enable offline access via **Google Drive → Settings → Offline** if you want to work without a connection.

**Either is sufficient to complete this course.** If you want to follow the "both, side by side" recommendation from Lecture 1, both are free to access with no purchase required.

## Reference sheets worth bookmarking

- **Microsoft's printable Excel shortcut cheat sheet** (linked from the keyboard shortcuts page above) — worth keeping open in a second tab or printed next to your keyboard for the first two weeks of this course.
- **Google's shortcut overlay, built into the app:** press `Ctrl+/` (Windows/ChromeOS) or `Cmd+/` (Mac) inside any open Sheet to bring up a searchable, categorized shortcut list without leaving the browser.

## Tools to install (optional, but useful all course long)

- **A second, distraction-free browser tab or window** for whichever app you're *not* using as primary — Lecture 1 recommends trying every major action in both apps at least once.
- **Nothing else is required this week.** Later weeks (Power Query, macros/VBA, Apps Script) will call out specific installs when they're needed — this week is deliberately tool-light so the focus stays on fundamentals.

## A note on versions

This course targets **Excel 365 / Excel 2021 or newer** and the **current web version of Google Sheets**. Menu paths described in the lectures (e.g., "View → Freeze Panes") match these versions; older versions (Excel 2016 and earlier) have the same underlying features but sometimes different ribbon locations — if a menu path doesn't match what you see, search the exact feature name in your version's Help search box (`Alt+Q` in Excel, or the Sheets Help menu) rather than assuming the feature is missing.
