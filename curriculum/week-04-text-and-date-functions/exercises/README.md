# Week 4 — Exercises

Three exercises, meant to be done in order, roughly matching the three lectures. Together they take you from splitting one messy column to cleaning a whole imported table to doing real date math a business would actually rely on.

## How to work these

- Use the same workbook you set up in the [week README](../README.md) (`crunch-week4.xlsx` or its Google Sheets equivalent) — add a new sheet per exercise so nothing overwrites your lecture work.
- Do them **in order**. Exercise 2 assumes the cleaning functions from Lecture 2; Exercise 3 assumes the date functions from Lecture 3.
- Type the seed data by hand rather than pasting — for this week specifically, typing forces you to notice exactly where the messiness lives (a stray space, an inconsistent format) instead of it being invisible to you the way it is when you paste.
- Every exercise expects a **formula-driven** solution, not a manually retyped/corrected value. If you find yourself fixing a cell by typing the clean value directly instead of writing a formula that produces it, stop and rewrite it as a formula — that's the actual skill being practiced.
- When a task says "Expected," check your result against it before moving on.
