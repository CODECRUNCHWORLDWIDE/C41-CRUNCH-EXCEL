# C41 · Crunch Excel — Syllabus

**Format:** 12 weeks · self-paced · open-source (GPL-3.0). Each week = 3 lectures, 3 exercises, 2 challenges, a mini-project, homework, and a quiz.

**Prerequisites:** You can open a spreadsheet app. No prior Excel, Sheets, or formula knowledge assumed — this course goes from your first cell to expert.

**Assessment (honor-based):** tick each lesson complete in the reader; finish the weekly mini-project and quiz. A certificate is issued on 100% completion to One-Stop members.

| Week | Theme | Mini-project |
|------|-------|--------------|
| 1 | Foundations + navigation (cells, rows, sheets, formatting, shortcuts) | Rebuild a formatted monthly budget from a blank sheet |
| 2 | Formulas + references (relative/absolute/mixed, operators, order) | Build a self-calculating invoice with a locked tax rate |
| 3 | Logical + lookup (IF/IFS, XLOOKUP, INDEX/MATCH, IFERROR) | Grade a class and price-match a product catalog |
| 4 | Text + date functions (parse, split, join, date/time math) | Normalize a messy contact + orders export |
| 5 | Data cleaning + validation (dedupe, trim, rules, drop-downs) | Turn a raw CSV dump into a clean, validated dataset |
| 6 | Tables + structured references (auto-expand, named formulas) | Convert a report to Tables driving a summary sheet |
| 7 | Pivot tables + charts (fields, grouping, calculated fields) | Answer 15 sales questions with pivots + charts |
| 8 | Dashboards + visualization (slicers, KPIs, layout, sparklines) | Build a one-screen interactive sales dashboard |
| 9 | Financial + what-if modeling (NPV/IRR, PMT, Goal Seek, scenarios) | Build a loan + investment model with scenarios |
| 10 | Dynamic arrays + modern functions (FILTER/SORT/UNIQUE, LET, LAMBDA) | Rebuild a report as one spilling dynamic-array formula |
| 11 | Power Query + data connections (import, transform, merge, refresh) | Build a repeatable ETL pipeline from multiple files |
| 12 | Capstone — automation (macros/VBA + Apps Script) + dashboard | Ship a self-refreshing automated dashboard |

**Outcome:** clean, model, and visualize real data — and ship an automated, self-refreshing spreadsheet dashboard in Excel or Google Sheets.
